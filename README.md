# 🧺 똑장 🧺

> 효율적 장 보기를 위한 장바구니 최적화 서비스

<br />

## 📝 소개

이 저장소는 프로젝트 '똑장'의 FE 저장소입니다.

이 프로젝트는 2025년 봄 학기 중앙대학교 소프트웨어대학 소프트웨어학부 캡스톤디자인(2) 과정에서 산출된 프로젝트입니다. Android 모바일 디바이스를 타겟으로 개발되고 있어요.

<br />

## ⚙ 스택

### 프레임워크

- **Jetpack Compose** | 앱 개발

### 주요 라이브러리

- **Naver Maps Android SDK** | 네이버 지도 사용
- **Coil** | 비동기 이미지 사용용
- **Retrofit2** | 고수준 API 요청
- **Hilt** | 의존성 주입 (DI)
- **Proto Datastore** | Google Protocol Buffer 기반 로컬 데이터 저장소
- **OkHttp** | HTTP 요청
